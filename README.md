# 🌍 DisasterPrep

**Offline-First Disaster Preparedness & Survival Inventory App**

DisasterPrep is a mobile application built using **React Native and Expo** that helps users prepare for disasters, manage survival inventory, and access emergency tools — even without internet connectivity.

---

## 🚀 Overview

- Offline-first architecture
- Survival inventory tracking
- Disaster awareness (India & Japan)
- Weather monitoring (temperature, humidity, wind)
- Emergency tools (SOS, contacts, location sharing)
- Preparedness scoring system
- Built fully from scratch

---

## 📱 Core Sections

- Onboarding (language, household size, contacts, location)
- Home (weather, alerts, guides, preparedness score)
- Emergency (SOS, contacts, location sharing)
- Inventory (items, expiry tracking, custom items)

---

## 🛠️ Tech Stack

- React Native
- Expo
- Zustand
- Open-Meteo API
- USGS API
- NASA EONET
- Local device storage

---

## ⚠️ Limitations

- Disaster data limited to **country/state level (India & Japan)**
- No district-level accuracy
- No backend (fully offline storage)
